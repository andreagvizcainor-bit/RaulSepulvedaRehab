# Sitio web - Clínica de Medicina Física y Rehabilitación

Este es un sitio estático en español, responsive, con las páginas:
- Inicio
- Servicios
- Sobre mí
- Contacto

Incluye botón de "Agendar por WhatsApp" (placeholder), sección de ubicación y horarios.

## Cómo iniciar el sitio localmente

1. Abre una terminal y navega al directorio del proyecto:
   ```bash
   cd c:\Users\drrau\codex\website
   ```
2. Instala dependencias (requiere Node.js y npm):
   ```bash
   npm install
   ```
3. Inicia el servidor local:
   ```bash
   npm run start
   ```
   El sitio quedará disponible en http://localhost:8080

> También puedes abrir `index.html` directamente en el navegador o usar `python -m http.server` si no tienes Node.
